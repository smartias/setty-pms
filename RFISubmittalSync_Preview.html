RFI / Submittal Tracker

Construction-Administration RFIs & submittals — log by hand or pull them in from Procore / Autodesk Forma. Saves to the same place as the main PMS.

▾ ℹ️ How this works (click to collapse)

  1. Pick your project. Only jobs marked "In Construction Administration" appear. A ⚡ means it has Procore/Forma emails ready to pull in.
  2. See everything logged. Click any RFI or submittal row to open and edit it. Use + Add to log a new one by hand (for items that don't come through Procore/Forma — e.g. emailed or PDF transmittals).
  3. Pull in Procore / Forma items. New ones show in the amber 📥 To import strip — tick the ones you want and click Import. They become normal RFIs/submittals you can then edit.
Put your name in the “You” box (top-right) so the log shows who added or changed each item. Safe by design: it never overwrites someone else's edit (it asks you to refresh instead), and importing the same item twice does nothing.
